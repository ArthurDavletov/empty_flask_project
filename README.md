# empty_flask_project
Пустой Flask-сервер

## Установка

```shell
python -m venv .venv
```

Windows:

```shell
.venv/Scripts/activate
```

Linux:

```shell
source .venv/bin/activate
```

## Установка зависимостей

```shell
pip install -r requirements.txt
```

## Запуск

```shell
python app.py
```

## Проверка

http://127.0.0.1:5000
